# Komodoro Timer

Komodoro Timer is a modern, intuitive Pomodoro timer application built for Android using Jetpack Compose. It helps users maintain focus and productivity by implementing the Pomodoro Technique, offering features like customizable timers, statistics, and a sleek Material 3 design.

## Tech Stack

This project uses the latest modern Android development tools and libraries:

-   **Language**: Kotlin
-   **UI Toolkit**: Jetpack Compose components with **Material 3** design system.
-   **Local Storage**: Room Database for persistent data storage.
-   **Charts**: [Vico](https://github.com/patrykandpatrick/vico) for data visualization.
